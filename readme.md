# rocketseat_ignite_ch03
- Challange 003 of the Ignite training by Rocketseat
- It's basically an Rest API that provides services add entities called repositories and record how many likes each repo have.
- The full challange description can be found [here](https://www.notion.so/Desafio-03-Corrigindo-o-c-digo-c15c8a2e212846039a367cc7b763c6dd)
- Template used can be found [here](https://github.com/rocketseat-education/ignite-template-corrigindo-o-codigo)

## Tech/framework used
- NodeJS
- ExpressJS
- Yarn
- Nodeman
- Jest

## Running
To run this sample use:

``` yarn dev ```

To test it use:

``` yarn test ```
